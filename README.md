# وجبات رمضان أبوظبي 🍽️

تطبيق ويب متكامل لطلب الوجبات الرمضانية في أبوظبي مع تصميم متجاوب وسلس للهواتف.

## ✨ المميزات

### 📱 تصميم متجاوب وسلس
- تصميم مخصص للهواتف أولاً (Mobile-First)
- تحميل سريع وبدون تأخير
- تفاعلات ناعمة وحركات سلسة
- دعم كامل للشاشات الصغيرة والكبيرة

### 🛒 نظام سلة متقدم
- سلة منبثقة من الأسفل للهواتف
- تحديث فوري للكميات والأسعار
- حفظ السلة تلقائياً
- منع الاختلاط بين مطاعم مختلفة

### 🔍 بحث وفلترة ذكي
- فلترة حسب مناطق أبوظبي
- بحث في أسماء المطاعم والوجبات
- عرض نتائج فوري أثناء الكتابة

### 💳 نظام طلب سهل
- 3 خطوات بسيطة لإتمام الطلب
- تأكيد البيانات قبل الإرسال
- شرح واضح لطريقة الدفع البنكي
- إرسال تلقائي لبيانات الدفع عبر واتساب

### 📊 حفظ الطلبات
- حفظ الطلبات في LocalStorage
- إمكانية التوسع لحفظ في Google Sheets
- تتبع حالة الطلبات

## 🚀 التنصيب السريع

### على GitHub Pages:
1. انشئ مستودع جديد على GitHub
2. ارفع جميع الملفات
3. اذهب إلى Settings → Pages
4. اختر فرع main كمصدر
5. انتظر دقائق ثم افتح الرابط

### على Render:
1. سجل في [render.com](https://render.com)
2. New + → Static Site
3. ربط بمستودع GitHub
4. اضغط Deploy

## ⚙️ التكوين

### تعديل بيانات المطاعم:
افتح ملف `script.js` وغير مصفوفة `restaurantsData` في الأعلى.

### تعديل معلومات التواصل:
افتح ملف `index.html` وغير:
- أرقام الهواتف في الفوتر
- رسوم التوصيل في `script.js` سطر 441
- بيانات البنك لكل مطعم
